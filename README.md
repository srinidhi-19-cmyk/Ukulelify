# 🎵 Ukulele India

A ukulele learning app built specifically for Indian musicians.
Find chords for Bollywood and Telugu songs, paste a YouTube link to get 
chord progressions, hum a tune to identify chords, and get real-time 
finger placement feedback — all in one place.

---

## Why This Exists

Apps like Ultimate Guitar and Chordify have poor coverage for Indian songs.
Yousician is expensive and not ukulele-focused.
This app is built for beginners in India who want to learn ukulele 
using songs they actually know.

---

## Features

- 🎸 **India Song Library** — Bollywood and Telugu songs with chords and progressions
- 📺 **YouTube Link Sync** — Paste any YouTube link, get chords synced to timestamps
- 🎤 **Hum-to-Chord** — Hum a tune, get the chords you need to play it
- 🎙️ **Vocal Isolation** — Hear only the singer, play ukulele alongside
- 👆 **Finger Placement Feedback** — Real-time feedback using hand tracking
- 🔁 **Chord Simplifier** — Simplifies any song to chords a beginner can play
- 🎼 **Chord Diagram Visualizer** — Animated fretboard with finger positions
- 🔧 **Ukulele Tuner** — Tune your ukulele using your microphone
- 📈 **Progress Tracker** — Track songs learned and practice streaks
- ⏰ **Practice Reminders** — Daily reminders to keep you consistent

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React.js |
| Backend | Python (FastAPI) |
| Audio Analysis | basic-pitch (Spotify), Web Audio API |
| Vocal Isolation | Spleeter (Deezer) |
| Hand Tracking | MediaPipe |
| Database | Firebase / Supabase |
| Deployment | Vercel (frontend), Render (backend) |

## ML & AI Integration

| Feature | Model / Library |
|---|---|
| Hum-to-chord detection | Spotify basic-pitch |
| Chord recognition from audio | basic-pitch |
| Vocal isolation | Spleeter by Deezer |
| Finger placement detection | MediaPipe Hands |
| Pitch detection (tuner) | Web Audio API autocorrelation |
